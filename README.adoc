The differences from the Khronos sample are as follows:

* Vulkan 1.3 and later versions now run on Vulkan 1.2 + Extension. This means it also works on Mac. The Extension is loaded with volk.
* Modified to use Vk-Bootstrap.
* Modified to use VMA (Vulkan Memory Allocator).
* Added the ability to display arbitrary shapes in the scene by adding nodes.
